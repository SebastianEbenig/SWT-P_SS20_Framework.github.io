Detailed Description

This class represents any container of cards with slots.

It provides functionality for loading, adding and removing cards from the container. The cards are not stored in the container itself, but in the card slots. Any additional effects that happen upon changing the container (for example database changes) can be implemented in the AdditonalAddUpdate() and AdditionalRemoveUpdate. A container generates its own slots if needed and can have a start slot amount and max slot amount.
