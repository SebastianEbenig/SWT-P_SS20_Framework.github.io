enum Framework.Scripts.RoomSystem.PlayerLobbyOptions

This enum represents the different options a game can have for its room players.

These options are handled in the framework, because they need to be specified before the game starts. This can be for example that each room player can bet a card.

Every new option is usually connected to the implementation of a new RoomGUI and RoomOverviewGUI to support the added functionalities. <author>John Beinecke</author>

Enumerator
NORMAL 

The game lobby has only the basic features hosting, joining, leaving and toggling the ready state.

CARD_OFFER 

The game lobby has the basic features and additional functionality for offering cards.

The winner of the game will receive all offered cards.
