Detailed Description

This class represents an interface for interacting with the GUI of a room that has the game lobby option "cardOffer".

It provides functionality for loading the Room GUI and for creating the RoomPlayerContainers that contain player data. It adds additional functionality for checking if the player has enough cards in their deck and inventory.

A detailed description for the different GameLobbyOptions can be found in GameLobbyOptions.cs .
