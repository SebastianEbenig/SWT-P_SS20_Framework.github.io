Detailed Description

This class represents a container for all information about cards and cardSets.

It loads this information from the database. The information is loaded in this order:

  1. CardSet information
  2. Card information
  3. Images for cards (locally) The information of a card can be loaded by using GetCardById(). It is implemented as a singleton.

    John Beinecke
