Detailed Description

This class represents an interface for interacting with the GUI of a room that has no additional features.

It provides functionality for loading the Room GUI and for creating the RoomPlayerContainers that contain player data.

A detailed description for the different GameLobbyOptions can be found in GameLobbyOptions.cs .
