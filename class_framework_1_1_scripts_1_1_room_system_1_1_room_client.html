Detailed Description

This class represents a players interface for interacting with the room system.

It provides functionality for cross-network communication and creating, joining, leaving and updating rooms. Rooms, RoomPlayers and playerInfos are only created on the Server which then are updated on all clients through the RpcUpdateRooms function. <author>John Beinecke</author>
